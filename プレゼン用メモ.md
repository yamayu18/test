# 百花ジェル ECサイト制作

## 制作のポイント

### 1. レスポンシブデザイン
- PCとスマホの両方で快適に閲覧できるよう、画面サイズに応じてレイアウトを最適化
- スマホでは画像が切れないよう、全体が表示される設計
- 3つの特徴セクションは、スマホでも横一列で見やすく配置

### 2. 使いやすさを重視
- 商品画像カルーセルは、指でスワイプして簡単に操作可能
- サムネイルを8枚すべて表示し、全体像が一目で分かる
- カート機能で、購入前に商品と数量を確認できる

### 3. 医療法人らしい信頼感
- シンプルで読みやすいデザイン
- 余計な装飾を避け、商品情報に集中できる構成
- 「売るためではなく、必要だったから作りました」というメッセージを前面に

### 4. 購入フローの最適化
- 通常購入と定期購入を明確に分けて表示
- 定期購入は約33%OFFで、初回の本数選択が簡単
- 送料無料を分かりやすく表示

### 5. 技術的な工夫
- 静的HTMLで軽量・高速表示
- GitHub Pagesで簡単に公開・更新可能
- スマホでの操作性を重視したUI設計

---

**シンプルで使いやすく、信頼感のあるECサイトを目指しました。**
